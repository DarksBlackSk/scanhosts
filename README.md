# scanhosts

* `scanhosts.sh` es una script en bash que he disenado para escanear una red en busca de dispositivos activos

## USO

* Si ejecutamos el script sin ningun parametro nos dara automaticamente las instrucciones de uso

<img width="1217" height="156" alt="image" src="https://github.com/user-attachments/assets/b2cb2cfd-8957-4c5a-ba39-48b049026ffb" />

* Asi que solo debemos pasarle como parametro una ip de la red que queramos escanear

```bash
./scanhosts.sh 20.20.20.2
```
